# 脏小猪（Drecksau）规则

## 概述

脏小猪（原名 Drecksau）是由 Frank Bebenroth 设计、KOSMOS 出版的快速卡牌游戏。每位玩家面前有一排干净的小猪，目标很简单：率先让自己所有的小猪都变脏！打出泥巴牌弄脏小猪，用谷仓保护它们，同时用雨水冲洗对手的小猪。

## 组件

- 小猪牌（双面卡：一面干净、一面脏）
- 泥巴牌
- 雨水牌
- 谷仓牌
- 闪电牌
- 避雷针牌
- 农夫牌（给小猪洗澡）
- 谷仓门牌（锁住谷仓）

## 准备

1. 根据人数分配小猪牌：
   - **2 人局**：每人 5 只小猪
   - **3 人局**：每人 4 只小猪
   - **4 人局**：每人 3 只小猪
2. 将所有小猪牌**干净面朝上**放在每位玩家面前。
3. 将其余卡牌洗匀，每位玩家发 **3 张**手牌。
4. 剩余牌面朝下作为抽牌堆。

## 游戏流程

轮到你时，**必须打出 1 张**手牌，然后从抽牌堆**抽 1 张**。

### 卡牌效果

| 卡牌 | 效果 |
|------|------|
| **泥巴** | 将自己的一只干净小猪翻到**脏面** |
| **雨水** | 所有未受保护的脏小猪（所有玩家）被冲洗回**干净面** |
| **谷仓** | 放在自己的一只小猪上。保护该小猪免受雨水影响 |
| **闪电** | 摧毁对手的一个谷仓（移出游戏） |
| **避雷针** | 附加到自己的谷仓上。永久保护该谷仓免受闪电攻击 |
| **农夫** | 农夫给一只脏小猪洗澡——即使在谷仓里也逃不掉！ |
| **谷仓门** | 附加到自己的谷仓（谷仓内必须已有脏小猪）。保护小猪免受农夫影响 |

### 特殊规则

- 如果你的 3 张手牌**都无法出牌**，向所有玩家展示手牌后全部弃掉，重新抽 3 张。
- 一只小猪同时拥有谷仓、谷仓门和避雷针时**完全安全**，不受任何威胁。
- 对对手使用的卡牌（雨水、闪电、农夫）使用后弃掉。
- 抽牌堆用完时，将弃牌堆洗匀形成新的抽牌堆。

## 获胜条件

**率先**让自己所有小猪都翻到脏面的玩家立即获胜！
