# 爆炸猫：黑盒版（NSFW Edition）规则

## 概述

黑盒版（官方名"NSFW Edition"）是爆炸猫的成人限定版本。**规则与红盒版（原版）完全相同**——仅将卡牌插画和风味文本替换为成人向、恶搞风格的荒诞插画。

这是一个独立游戏，支持 **2-5 人**游玩（不需要红盒原版）。也可以和红盒版混合，最多支持 **9 人**游玩。

## 红盒 vs 黑盒

| | 红盒（原版） | 黑盒（NSFW） |
|---|---|---|
| **规则** | 标准 | 完全一致 |
| **卡牌数量** | 56 张 | 56 张 |
| **玩家人数** | 2-5 人 | 2-5 人 |
| **画风** | 全年龄友好 | 成人向/恶搞 |
| **建议年龄** | 7+ | 17+ |
| **可混合** | 可以，最多 9 人 | 可以，最多 9 人 |

## 快速规则回顾

1. **准备：** 取出所有爆炸猫牌和拆弹牌。每人发 1 张拆弹牌 + 7 张随机牌。将剩余拆弹牌和爆炸猫牌（比玩家人数少一张）洗入抽牌堆。

2. **你的回合：** 可以打出任意数量的手牌（或不出），然后**抽一张牌**结束回合。

3. **如果抽到爆炸猫：** 必须打出拆弹牌否则被淘汰。拆弹成功后，秘密地将爆炸猫放回抽牌堆的任意位置。

4. **最后存活的玩家获胜。**

## 卡牌类型

| 卡牌 | 效果 |
|------|------|
| **拆弹牌** | 从爆炸猫中拯救你 |
| **攻击牌** | 不抽牌结束回合；下一位玩家必须连续行动 2 次 |
| **跳过牌** | 不抽牌直接结束回合 |
| **索取牌** | 强制另一位玩家给你一张他选择的牌 |
| **洗牌牌** | 随机打乱抽牌堆 |
| **预见未来** | 偷看抽牌堆顶部 3 张牌 |
| **否决牌（Nope）** | 取消其他玩家打出的任何功能牌（拆弹牌和爆炸猫除外） |
| **猫咪牌** | 收集相同的对子/三条来偷取对手的牌 |

> 完整详细规则请参阅**爆炸猫**（红盒版）本体规则页面。
