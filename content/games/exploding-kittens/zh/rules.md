# 爆炸猫（Exploding Kittens）规则

## 概述

爆炸猫是一款充满策略性的猫咪版俄罗斯轮盘游戏。玩家轮流抽牌，直到有人抽到爆炸猫——此时该玩家出局，除非手中有拆弹牌。最后存活的玩家获胜。

## 组件

- **56 张卡牌**，包括：
  - 4 张爆炸猫
  - 6 张拆弹牌
  - 各种行动牌（攻击、跳过、预见未来等）
  - 否决牌
  - 猫咪牌（用于组合技）

## 准备

1. 从牌堆中取出所有爆炸猫（4 张）和拆弹牌（6 张）。
2. 给每位玩家发 1 张拆弹牌。
3. 将剩余的拆弹牌洗回牌堆。
4. 给每位玩家发 7 张牌。
5. 将爆炸猫放回牌堆，数量为玩家人数减 1（如 3 人局放 2 张爆炸猫）。
6. 移除剩余的爆炸猫。
7. 洗匀牌堆并面朝下放置。

## 游戏流程

轮到你的回合时，可以打出任意数量的牌（或不打），然后**必须**从牌堆顶部抽 1 张牌。

如果抽到爆炸猫：
- 你**死亡**并出局
- 除非你有**拆弹牌** — 使用它，将爆炸猫秘密放回牌堆中任意位置（包括牌堆顶）

## 卡牌参考

| 卡牌 | 效果 |
|------|------|
| **爆炸猫** | 必须用拆弹牌拆除，否则死亡 |
| **拆弹牌** | 抵挡一次爆炸猫。将其秘密放回牌堆 |
| **攻击** | 结束回合不抽牌。下家必须连续执行 2 个回合 |
| **跳过** | 结束回合不抽牌 |
| **预见未来** | 偷看牌堆顶部 3 张牌 |
| **洗牌** | 重新洗牌堆 |
| **索要** | 指定一名玩家，他必须给你 1 张自选手牌 |
| **否决** | 取消任意行动牌的效果（包括否决牌）。可随时打出 |
| **猫咪牌**（5 种） | 凑成一对可随机抢 1 张手牌。凑成三张可指名要 1 张牌 |

## 猫咪牌组合技

- **两张相同** — 从任意玩家手中随机偷 1 张牌
- **三张相同** — 指定一张牌名，从任意玩家手中拿走（如果他有）
- **五张不同** — 从弃牌堆中拿回任意 1 张牌

## 策略提示

- 保留拆弹牌 — 它是你唯一的保命手段
- 牌堆较少时先用预见未来再抽牌
- 否决牌可以在关键时刻救命
- 猫咪牌留到组合使用时更强

## 获胜条件

最后一位没有爆炸的玩家获胜。
