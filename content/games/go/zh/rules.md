# 围棋规则

## 游戏概述

围棋是一种起源于中国、拥有 2500 多年历史的双人抽象策略棋类游戏。玩家轮流在网格棋盘上放置黑白棋子，目标是围住比对手更多的地盘。围棋规则简单，却被公认为世界上最深奥的策略游戏之一。

## 游戏组件

- 围棋棋盘：19×19 线交叉点（初学者可使用 13×13 或 9×9）
- 181 颗黑子
- 180 颗白子
- 棋罐

## 游戏准备

1. 将棋盘放在两位玩家之间。
2. 执黑者拿黑子，执白者拿白子。
3. 棋盘初始为空。
4. 黑方先行。

## 游戏流程

### 落子

玩家轮流将一颗己方颜色的棋子放在任意空交叉点上。棋子放在**交叉点**上，而非方格内。

- 棋子一旦放置，不可移动（除非被提走）。
- 可在任何时候**停着**（pass）。连续两次停着，棋局结束。

### 气和提子

一颗或一组相连的棋子拥有**气**——即该棋子/棋组正交方向（上下左右，不含斜对角）相邻的空交叉点数量。

- 当一组棋子**气数为零**时，即被提走并从棋盘上移除。
- 被提走的棋子作为俘虏保留，终局时计入分数。

**示例：**
```
● ○
● ○
```
如果黑棋堵上白棋最后一口气，白子即被提走。

### 劫

**劫**是指单颗棋子可以无限来回提子的局面。**劫规则**禁止立即回提——必须先在其他地方走一手，待对方应一手后，方可回提。

### 死活

- **两眼活棋**：如果一个棋组拥有（或可以做出）两个独立的眼（空点），对方无法同时填满，则该棋组**活**。
- **死棋**：无法做出两只眼的棋组为**死棋**，终局时作为俘虏移除（中国规则）或留着证明可提（日本规则）。

### 地盘

**地盘**（空）是指被己方棋子围住的空交叉点。终局时：
- **中国规则（数子法）**：己方活子 + 己方围住的空点。
- **日本规则（数目法）**：己方围住的空点 + 提走的俘虏。

## 游戏结束

当双方连续停着时，对局结束：

1. 双方确认哪些棋组为死棋，作为俘虏移除。
2. 计算各方得分：
   - **数子法（中国）**：己方活子 + 己方空点。
   - **数目法（日本）**：己方空点 + 提子俘虏。
3. **贴目**：白方获得 **6.5 目**（中国规则）或 **6.5 目**（日本规则）作为后手补偿。0.5 目杜绝平局。

**得分高者获胜。**

## 让子制度

平衡棋力差距：
- 较弱方（持黑）在开局前于指定星位放置让子。
- 让子局白方先行。
- 让子数：2–9 子，黑方不贴目。

## 特殊规则

- **自杀**：中国规则下，禁止下出导致己方棋组零气的着手（自填），除非同时提掉对方棋子。
- **双活（Seki）**：双方都无法落子而不危及己方棋组的状态。双方棋组皆活但无两只眼。双活中的空点为公气，不计入任何一方的分数。
- **单官**：地盘之间不影响计分的空点。数棋前先填单官。
