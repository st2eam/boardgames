# 自然和弦 规则

## 概述

**《自然和弦》**（Harmonies）是由设计师 Johan Benvenuto 创作、**Libellud**（艾赐魔袋集团）于2024年出版的一款板块放置策略桌游。玩家通过在个人版图上放置彩色地形标记，构建3D立体生态景观，并在地形满足动物卡的栖息地条件时放置动物方块得分。

## 组件

- **1 张中央版图** — 双面（单人面 / 2–4人面）
- **4 张个人版图** — 双面（A面：河流 / B面：岛屿）
- **120 个木质标记** — 23个蓝色（水域）、23个灰色（山脉）、21个棕色（树干）、19个绿色（树冠）、19个黄色（原野）、15个红色（建筑）
- **32 张动物卡** — 每张展示栖息地图案和方块放置位
- **10 张自然精灵卡** — 可选进阶玩法
- **75 个动物方块** + 4个自然精灵方块
- **1 个布袋** — 用于抽取标记
- 4张提示卡和计分纸

## 准备

1. **中央版图** — 放在中间，2–4人面朝上（单人游戏使用单人面）。
2. **标记** — 将所有标记放入布袋混合均匀。从袋中随机抽3个标记，依次放到中央版图的5个空格上。
3. **动物卡** — 洗牌后翻开5张面朝上放在中央版图旁，其余形成摸牌堆。
4. **方块** — 将所有动物方块放在伸手可及处。
5. **个人版图** — 每位玩家取一张个人版图，所有人必须使用**同一面**（新手推荐A面）。
6. **起始玩家** — 最近看过壮丽风景的玩家先手。将中央版图的箭头指向起始玩家。

## 回合流程

在自己的回合中，按任意顺序执行以下行动。一项为**必须**，两项为**可选**：

| 行动 | 类型 |
|---|---|
| **拿取并放置标记** | 必须 |
| **拿取动物卡** | 可选（每回合至多1张） |
| **放置动物方块** | 可选（无次数限制） |

回合结束时：从袋中补3个标记到空出的中央版图空格，补满动物卡展示区至5张。

---

### 1. 拿取并放置标记（必须）

选择中央版图上的**一个空格**，拿走上面的全部3个标记。本回合内**必须将三个标记全部放置**到你的个人版图上。你可以在放置标记之间穿插可选行动。

#### 堆叠规则

标记可堆叠至多 **3层**：

| 标记 | 可放置在 | 上面可放置 |
|---|---|---|
| 🔵 **蓝色（水域）** | 仅空格 | 无 |
| 🟡 **黄色（原野）** | 仅空格 | 无 |
| ⚫ **灰色（山脉）** | 空格或其他灰色 | 仅其他灰色 |
| 🟤 **棕色（树干）** | 空格或其他棕色 | 棕色或绿色 |
| 🟢 **绿色（树冠）** | 仅1或2个棕色上 | 无 |
| 🔴 **红色（建筑）** | 灰色、棕色或红色上（下方仅1个标记） | 无（不能作为第3层） |

> 一旦该格放置了动物方块，则不能再在上面堆叠标记。

---

### 2. 拿取动物卡（可选）

- 从5张面朝上的动物卡中选择**1张**，放在个人版图上方。
- **最多持有4张**动物卡。
- 取卡牌右下角显示的数量的动物方块，按**从下到上**的顺序放在卡牌的空位上。

---

### 3. 放置动物方块（可选）

当个人版图上的标记图案与某张动物卡的栖息地条件**完全匹配**时：

- 从该卡上拿取**最下方的可用方块**。
- 将其放在图案中对应的一个标记上。
- 图案可**任意旋转或镜像**匹配。
- 树/山的高度必须**完全匹配**。
- 图案中的**红色（建筑）**标记可以是任意有效的建筑类型。
- 一个标记可参与多个动物图案，但只能容纳**一个**方块。
- 方块一旦放置就锁定——即使后来堆叠破坏了原先的图案也不影响。
- 当某张动物卡上的**所有方块都被移除**后，弃掉该卡——不再计入4张手牌上限。

---

## 游戏结束

满足以下任一条件时游戏结束：

1. 需要补充中央版图标记时**布袋已空**，**或**
2. 回合结束时，有玩家的个人版图空格**≤2格**。

所有玩家获得相同的回合数。若在轮次中触发结束，剩余玩家各进行最后一个回合。

## 计分

### 🌳 树木

| 构成 | 分数 |
|---|---|
| 1个棕色（无绿色） | 0 |
| 2个棕色（无绿色） | 0 |
| 1个绿色（灌木） | 1 |
| 1棕+1绿（树木） | 3 |
| 2棕+1绿（大树）| 7 |

### ⛰️ 山脉

*必须与至少一个其他山脉相邻才能计分。*

| 高度 | 分数 |
|---|---|
| 1层 | 1 |
| 2层 | 3 |
| 3层 | 7 |
| 孤立（无相邻山脉）| 0 |

### 🌾 原野

- 每组**2个及以上相邻黄色标记** = **5分**。
- 大组原野不会得更多分——应拆分为多个独立小群组。

### 🏠 建筑

- 建筑（红色叠在灰/棕/红色上）= **5分** — *前提是*与至少**3种不同颜色的标记**相邻。
- 每个相邻堆仅计算**最顶部的标记**颜色。

### 💧 水域 — A面（河流）

计算蓝色标记的最长连续链，取两端之间的**最短路径**：

| 链长 | 分数 |
|---|---|
| 1 | 0 |
| 2 | 2 |
| 3 | 5 |
| 4 | 8 |
| 5 | 11 |
| 6 | 15 |
| 超过6，每+1 | +4 |

> ⚠️ 环形惩罚——环状河流按最短路径计算约等于环上一半标记数量+1。

### 🏝️ 水域 — B面（岛屿）

- 每个被蓝色标记**分隔**的陆地区域 = **5分**。
- 你始终至少有1个岛屿。

### 🐾 动物卡

对于每张动物卡，计分 = 卡上**最上方空位**的数字。若未放置任何方块，该卡计 **0分**。

### 平分决胜

放置动物方块最多者获胜。仍平局则共享胜利。

## 自然精灵卡（可选变体）

建议熟悉基础规则后使用：

- **准备：** 每位玩家抽2张精灵卡，保留1张，另一张放回盒子。放在版图上方（计入4张手牌上限），放置一个精灵方块。
- **玩法：** 类似动物卡，有特殊的栖息地条件。放置方块后移开（如完成的动物卡）。
- **计分：** 两种类型：
    1. **景观计数型** — 每创造一个指定布局计分。
    2. **群组计分型** — 对每组符合条件的地形计分。

精灵卡分数**累加**到其他所有计分上。

## 单人模式

| 规则 | 变化 |
|---|---|
| **中央版图** | 使用单人面（3个空格，而非5个） |
| **动物卡** | 仅展示3张 |
| **标记清理** | 每回合开始时，移除中央版图上所有6个剩余标记（放回盒子而非布袋），然后补满3格 |
| **卡牌刷新** | 若未拿取动物卡，可弃掉1张面朝上的卡并从牌堆替换 |

### 单人计分 — 获得太阳 (🌞)：

| 分数 | 太阳 |
|---|---|
| 40+ | 1 🌞 |
| 70+ | 2 🌞 |
| 90+ | 3 🌞 |
| 110+ | 4 🌞 |
| 130+ | 5 🌞 |
| 140+ | 6 🌞 |
| 150+ | 7 🌞 |
| 160+ | 8 🌞 |

- 使用A面板图：**+1太阳**
- 未使用自然精灵卡：**+2太阳**
- 使用群组计分型精灵卡：**+1太阳**

## 策略提示

- **提前规划** — 根据展示区中的动物卡预判放置标记，但注意其他玩家可能抢先拿走。
- **拆分原野** — 多个小原野群组比一个大原野更有效率。
- **注意平衡** — 动物卡得分和地形得分可能冲突，寻找最优权衡。
- **山脉相连** — 孤立的山脉即使3层也得0分。
- **建筑需要色彩** — 将其放置在多种颜色标记环绕的位置。
- **避开环形水域** — A面尽量构建蜿蜒但不闭合的河流。
