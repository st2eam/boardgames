# 三国杀：风云际会 规则

## 概述

**《风云际会》**是游卡桌游于2025年推出的官方扩展包，集成了**三大热门变体模式**——用间篇、忠胆英杰、应变篇——每个模式都有独特的机制、卡牌和策略。

**需要基础版（三国杀标准版）才能游玩。**

---

## 模式一：用间篇

**用间篇**将间谍与谍战元素引入三国杀，核心新机制是**赠予**，配合20+张全新卡牌。

### 核心机制：赠予

牌面左下角标有**"赠"**字的卡牌可以赠予其他角色。

| 规则 | 说明 |
|---|---|
| **时机** | 出牌阶段 |
| **目标** | 任意一名其他角色 |
| **装备牌** | 直接进入目标装备区（替换原有装备） |
| **非装备牌** | 进入目标手牌 |
| **目标可拒绝？** | **不能**——赠予强制接受 |
| **可被阻止？** | 仅【战象】效果会使赠予失败 |

> 赠予与"交给"是**不同的**操作——赠予正面向上且不可拒绝。

### 新基本牌

| 卡牌 | 张数 | 效果 |
|---|---|---|
| **毒** | 5 | 从牌堆获得时必须**展示**并赠予他人。正面向上离开手牌时（赠予、弃置、重铸、拼点展示等），你**流失1点体力**。被顺手牵羊（背面）或通过"交给"类技能转移时**不触发**。 |
| **刺杀** | 10 | 【杀】的变体。被【闪】抵消时，目标须额外**弃1张手牌**，否则依然造成伤害。 |

### 新锦囊牌

| 卡牌 | 张数 | 效果 |
|---|---|---|
| **树上开花** | 2 | 对自己使用，弃置1~2张牌并摸等量牌。若弃置中包含装备牌，多摸1张。 |
| **刮骨疗毒** | 2 | 对一名**已受伤**角色使用。回复1点体力，然后你可无代价弃置其区域内一张【毒】。 |
| **推心置腹** | 2 | 对**距离为1**的一名角色使用。你获得其区域内至多2张牌，然后交给其等量手牌。 |
| **趁火打劫** | 3 | 展示目标一张手牌。对方选择：**交给你**，或**受你1点伤害**。 |

### 新装备牌

| 卡牌 | 类型 | 范围 | 效果 |
|---|---|---|---|
| **折戟** | 武器 | 0 | **无任何效果**。可赠予坑害敌人。 |
| **七星宝刀** | 武器 | 2 | 进入装备区时，**弃置你装备区和判定区的所有其他牌**。 |
| **倚天剑** | 武器 | 2 | 【杀】造成伤害后，可弃1手牌**回复1点体力**。 |
| **引蜂衣** | 防具 | — | **锁定技**：你受到的锦囊伤害+1，【毒】流失体力+1。 |
| **女装** | 防具 | — | **锁定技**：**男性角色**成为【杀】目标时判定，黑色则伤害+1。 |
| **驽马** | -1马 | — | **锁定技**：其他角色与你的距离**始终为1**，其他距离修改效果无效。 |
| **战象** | +1马 | — | **锁定技**：其他角色对你的赠予**一律失败**（牌进入弃牌堆）。 |
| **信鸽** | 宝物 | — | 出牌阶段限一次，将一张手牌交给一名其他角色。 |

### 策略要点

- 【毒】是核心博弈卡——在玩家间流转，正面离手即掉血。
- 赠予装备可坑害敌人：塞【折戟】废掉武器槽、塞【引蜂衣】使其易伤、塞【七星宝刀】炸装备。
- 【战象】是唯一防赠予的神器。

## 模式二：忠胆英杰

**忠胆英杰**（又称"明忠模式"）颠覆传统身份局：**开局公开一名忠臣身份（明忠），而主公潜伏于其余玩家之中。**设计初衷是解决标准8人局中"反贼杀"问题（反贼胜率过高）。

### 基本配置

| 项目 | 内容 |
|---|---|
| **人数** | 8人：1主公 + 2忠臣 + 4反贼 + 1内奸 |
| **限制** | 禁用点将卡、手气卡、换将卡。禁用邓艾、徐晃、于吉、左慈、所有神武将等。 |

### 核心规则

1. **身份与选将：**
   - 随机选一名忠臣作为**明忠**，公开身份。
   - **主公隐藏身份**，潜伏在其余7人中。
   - 明忠获得**7张选将框**（含崔琰、皇甫嵩两名专属武将）。
   - 明忠**体力上限+1**。

2. **行动顺序：**
   - 明忠固定为**1号位**，首回合开始。

3. **主公亮将：**
   - **明忠死亡后**，主公必须亮明身份并获得主公技。

### 击杀奖惩

| 情形 | 结果 |
|---|---|
| 任何人杀死反贼 | 摸3张牌 |
| 任何人杀死明忠（暗主除外）| 摸3张牌 |
| **暗主杀死明忠** | 不摸牌，**弃置所有牌** |
| 暗主杀死暗忠 | 无惩罚 |
| 明主杀死暗忠 | 弃置所有牌 |

### 忠臣技（明忠专属）

**洞察** — 适用于体力上限≤3的武将（加上限后）：
- 游戏开始时，随机查看一名**反贼**的身份牌。
- 准备阶段，可弃置场上任意区域的一张牌。

**舍身** — 适用于体力上限≥4的武将（加上限后）及所有女性武将：
- **锁定技**：主公濒死求桃完毕后仍处于濒死状态时，主公+1体力上限、回复体力至X点（X=你的体力值）、获得你所有牌，然后**你死亡**。
- 以命换主的终极保主手段。

### 专属武将

| 武将 | 体力 | 技能 |
|---|---|---|
| **崔琰** | 3 | **雅望**（锁定技）：摸牌阶段放弃摸牌，改为摸X张（X=与你体力值相等的角色数）。出牌阶段至多使用X张牌。**殉志**：准备阶段，若上家和下家体力值均与你不同，可失去1点体力，令手牌上限+2（永久叠加）。 |
| **皇甫嵩** | 4 | **奋钺**：出牌阶段限X次（X=存活忠臣数）。与一名角色拼点。若赢，选择：其本回合不能使用/打出手牌，或视为对其使用【杀】（不计次数）。若没赢，结束出牌阶段。 |

### 专属卡牌（9张）

| 卡牌 | 替换原牌 | 效果简述 |
|---|---|---|
| **声东击西** | 顺手牵羊 | 交给距离1以内角色1张手牌，其将2张牌给另一角色 |
| **弃甲曳兵** | 借刀杀人 | 目标弃置所有武器/-1马，或所有防具/+1马 |
| **草木皆兵** | 兵粮寸断 | 延时锦囊，判定非梅花则少摸1张，距离1角色各摸1张 |
| **金蝉脱壳** | 无懈可击 | 唯一手牌时免疫锦囊/基本牌并摸2张；被弃置时摸1张 |
| **增兵减灶** | 无中生有/五谷丰登 | 目标摸3张牌，然后弃1张非基本牌或弃2张牌 |
| **浮雷** | 闪电 | 判定黑桃则受X点雷伤（X=连续黑桃次数），然后移到下家判定区 |
| **七宝刀** | 青釭剑 | 范围2，无视防具，目标未受伤则伤害+1 |
| **衠钢槊** | 青龙偃月刀 | 范围3，杀指定目标后可互弃一张手牌 |
| **烂银甲** | 八卦阵 | 手牌当【闪】，受到【杀】伤害后弃置此防具 |

## 模式三：应变篇

**应变篇**是自2011年《军争篇》以来暌违十年的首次重大卡池革新，引入了全新的**应变**关键词系统、**冰**属性伤害、**卜算**机制以及**晋势力**的**隐匿**玩法。

### 新关键词：应变

带有**应变**关键词的卡牌在你满足特定条件时触发额外效果：

| 条件 | 含义 | 常见加成 |
|---|---|---|
| **助战** | 使用后，其他角色可弃置一张同类型手牌触发 | 目标+1 / 目标-1 / 伤害+1 / 摸1张 / 不可被响应 / 执行所有选项 |
| **空巢** | 使用此牌后手牌为0 | 摸牌 / 目标+1 |
| **富甲** | 使用此牌时手牌数全场最多 | 目标-1 / 不可被响应 |
| **残躯** | 使用此牌时体力值为1 | 目标+1 / 伤害+1 |

### 新伤害类型：冰

- 冰属性伤害可选择防止，改为**依次弃置目标两张牌**（一张一张弃）。
- 仅对直接目标生效——铁索传导**不触发**。
- 强力克制卖血流武将（郭嘉、曹操等）。

### 新机制：卜算

- **卜算X** = 观看牌堆顶X张牌，以任意顺序放回牌堆顶或牌堆底。
- 类似诸葛亮〖观星〗但作为卡牌效果。

### 新增卡牌

| 卡牌 | 类型 | 张数 | 效果 |
|---|---|---|---|
| **冰杀** | 基本 | — | 造成1点冰属性伤害。 |
| **随机应变** | 锦囊 | 2 | 视为本回合使用/打出的上一张基本牌或普通锦囊牌。 |
| **出其不意** | 锦囊 | 3 | 展示目标一张手牌，若花色与此牌不同，其受到1点伤害。 |
| **逐近弃远** | 锦囊 | 4 | 距离>1则弃置其区域内一张牌；距离=1则获得其区域内一张牌。 |
| **水淹七军** | 锦囊 | 3 | 目标选择弃置2张装备牌，或受到1点雷电伤害。 |
| **洞烛先机** | 锦囊 | 4 | 卜算2，然后摸两张牌。 |
| **五行鹤翎扇** | 武器 | 范围4 | 可将属性【杀】当任意其他属性【杀】使用。 |
| **乌铁锁链** | 武器 | 范围3 | 【杀】指定目标后，若其已横置则观看其手牌，若未横置则横置之。 |
| **黑光铠** | 防具 | — | **锁定技**：你成为【杀】或普通锦囊目标后，若非唯一目标，此牌对你无效。 |
| **护心镜** | 防具 | — | 受到致命伤害时，可弃置此装备防止该伤害。 |
| **铜雀** | 宝物 | — | 使用带有应变效果的牌时，无视条件直接生效。 |
| **天机图** | 宝物 | — | 进入装备区时弃一张其他牌；离开装备区时将手牌摸至5张。 |
| **太公阴符** | 宝物 | — | 出牌阶段开始时可横置一名角色；出牌阶段结束时可重铸一张手牌。 |

### 晋势力 — 隐匿机制

晋势力武将（司马家族）拥有独特的**隐匿技**：

> **隐匿状态**：游戏开始时，武将暗置。此状态下：无体力值、无技能展示、性别为男。受到扣减体力的伤害/流失，或自己回合开始时，翻转并亮出武将——称为**"登场"**，触发登场技能。

#### 晋势力武将

| 武将 | 体力 | 技能 |
|---|---|---|
| **司马懿** | 3 | **不臣**（隐匿技）：于其他角色回合登场后，可获得当前回合角色的一张牌。**鹰视**（锁定技）：出牌阶段，牌堆顶X张牌始终对你可见（X=体力上限）。**雄志**（限定技）：出牌阶段，可展示牌堆顶牌并直接使用，重复此流程直到无法使用。**权变**：出牌阶段首次使用/打出某花色手牌时，可从牌堆顶X张中获一张不同花色。本回合至多使用X张手牌。 |
| **张春华** | 3 | **慧识**：摸牌阶段可放弃摸牌，观看牌堆顶X张（X=牌堆数量个位数），获得其中一半，其余置底。**清冷**：角色回合结束时，若其体力+手牌数≥X，可将一张牌当无距离限制的冰【杀】对其使用。**宣穆**（隐匿技）：于其他角色回合登场时，防止受到的伤害直到回合结束。 |
| **司马师** | 3/4 | **韬隐**（隐匿技）：登场后令当前回合角色手牌上限-2直到回合结束。**夷灭**：造成伤害时可失去1体力令伤害增加；结算后目标回复相应体力。**睿略**（主公技）：其他晋势力出牌阶段可交给你一张【杀】或伤害锦囊。**泰然**（锁定技）：回合结束时体力补至上限、手牌补至上限；下回合出牌阶段开始时失去/弃置上回合补的体力与手牌。 |
| **夏侯徽** | 3 | **宝箧**（隐匿技）：登场后从牌堆/弃牌堆获得一张宝物牌并可使用。**宜室**：一名其他角色于出牌阶段弃置手牌后，你可令其获得其中一张，你获得其余牌。**识度**：出牌阶段限一次，与一名角色拼点，若赢则获得其所有手牌，然后将你一半手牌交给其。 |
| **司马昭** | 3 | **推弑**（隐匿技）：于其他角色回合登场后可令其对你指定的角色使用【杀】，否则对其造成1点伤害。**筹伐**：展示一名角色一张手牌，其手牌中不同类型的牌均视为【杀】直到回合结束。**昭然**：出牌阶段开始时，可使手牌对所有角色可见，本阶段失去某花色最后一张手牌时可摸牌或弃置他人牌。**成务**（主公技）：其他晋势力攻击范围内的角色均视为在你的攻击范围内。 |
| **王元姬** | 3 | **识人**（隐匿技）：于其他角色回合登场后，可对当前回合角色发动一次宴戏。**宴戏**：出牌阶段限一次，令一名角色随机一张手牌+牌堆顶2张混合展示，猜哪张来自其手牌。猜对获得3张；猜错获得选中的牌（本回合不计手牌上限）。 |
| **张虎&乐綝** | 4 | **袭爵**：游戏开始获4枚"爵"标记；回合结束获X枚（X=本回合造成伤害点数）。可弃置标记发动突袭或骁果。 |
| **杜预** | 4 | **三陈**：令一名角色摸3弃3，若弃置的牌种类均不同则摸1张并重置技能（本回合不能指定同一目标）。**诏讨**（觉醒技）：发动3次三陈后减1体力上限，获得**破竹**：可将手牌当【出其不意】使用，若未造成伤害则本回合技能无效。 |

### 牌堆变化

总牌数保持**161张**（与军争篇相同）。移除了火攻、寒冰剑、朱雀羽扇、方天画戟、白银狮子、借刀杀人等旧卡。属性伤害牌从17张增至25张。
