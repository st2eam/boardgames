# 麻将 - 通用入门规则

## 概述

麻将是一种起源于中国的传统桌上游戏，使用一套 144 张牌（麻将牌），通常由 4 名玩家参与。游戏目标是通过摸牌、打牌，最先将手中的牌组成合法的胡牌牌型。

## 牌具组成

### 序数牌（108 张）

| 花色 | 牌面 | 数量 | 示例 |
|------|------|------|------|
| 万子 | 一万～九万 | 各 4 张 | [1m][2m][3m][4m][5m][6m][7m][8m][9m] |
| 筒子 | 一筒～九筒 | 各 4 张 | [1p][2p][3p][4p][5p][6p][7p][8p][9p] |
| 条子 | 一条～九条 | 各 4 张 | [1s][2s][3s][4s][5s][6s][7s][8s][9s] |

### 字牌（28 张）

| 类别 | 牌面 | 数量 |
|------|------|------|
| 风牌 | [E][S][W][N] | 各 4 张 |
| 箭牌 | [C][F][B] | 各 4 张 |

### 花牌（8 张，部分规则不使用）

春、夏、秋、冬、梅、兰、竹、菊各 1 张。

## 游戏流程

### 1. 开局准备

1. **洗牌**：将所有牌面朝下混合洗匀
2. **砌牌**：每人取 34 张牌（不使用花牌时）砌成 17 墩（每墩上下 2 张）
3. **定庄**：通过掷骰子确定庄家（东风位）
4. **发牌**：庄家摸 14 张，其余各摸 13 张

### 2. 基本轮流

玩家按逆时针顺序轮流操作：

1. **摸牌**：从牌墙摸取 1 张牌
2. **整理**：考虑是否要打出哪张牌
3. **打牌**：选择 1 张不需要的牌打出

### 3. 特殊操作

当其他玩家打出一张牌时，你可能可以执行以下操作：

| 操作 | 条件 | 优先级 |
|------|------|--------|
| **胡** | 该牌能让你凑成胡牌型 | 最高 |
| **杠** | 你手中已有该牌 3 张 | 高 |
| **碰** | 你手中已有该牌 2 张 | 中 |
| **吃** | 你手中有与该牌相邻的 2 张序数牌（仅限上家打出的牌） | 低 |

## 胡牌条件

### 标准胡牌型

手牌需组成以下结构：

$$4 \times \text{面子} + 1 \times \text{雀头} = 14 \text{ 张}$$

**面子**（3 张一组）：
- **顺子**：同花色连续 3 张，如 [1m][2m][3m]（一二三万）
- **刻子**：相同的 3 张，如 [C][C][C]（三张红中）

**雀头**（对子）：相同的 2 张，如 [E][E]（两张东风）

### 特殊牌型

| 牌型 | 说明 |
|------|------|
| **七对** | 7 组对子 |
| **十三幺** | 一九万、一九筒、一九条、东南西北中发白各一张 + 其中任一张配对 |

### 自摸与放炮

- **自摸**：自己摸到的牌完成胡牌型
- **放炮（点炮）**：别人打出的牌让你完成胡牌型

## 面子类型详解

| 名称 | 组成 | 来源 |
|------|------|------|
| 顺子 | 同花色连续三张 | 吃牌或自摸 |
| 刻子 | 三张相同 | 碰牌或自摸（暗刻） |
| 杠子 | 四张相同 | 明杠/暗杠/加杠 |

### 杠的分类

- **暗杠**：手中自摸凑齐 4 张，面朝下亮出
- **明杠**：手中 3 张 + 别人打出 1 张
- **加杠**：已碰的 3 张 + 自摸第 4 张

杠后需从牌墙末端补摸 1 张牌。

## 常见术语

| 术语 | 解释 |
|------|------|
| 听牌 | 只差一张就能胡的状态 |
| 门清 | 没有吃碰杠（全部暗手） |
| 断幺 | 手牌中没有一、九和字牌 |
| 平胡 | 四组顺子 + 一对雀头 |
| 清一色 | 全部为同一花色 |
| 混一色 | 一种花色 + 字牌 |
| 对对胡 | 全部由刻子/杠子 + 雀头组成 |
| 牌墙 | 砌好的未发牌堆 |
| 海底 | 牌墙最后一张牌 |

## 计分入门

不同地区的麻将计分规则差异很大。通用的基础概念：

| 概念 | 说明 |
|------|------|
| 底分 | 每局基础分值 |
| 番 | 倍数，特殊牌型可叠加 |
| 自摸加倍 | 自摸胡通常得分翻倍 |
| 庄家加倍 | 庄家胡/被胡时分值更高 |

### 常见番数参考

| 牌型 | 参考番数 |
|------|----------|
| 平胡 | 1 番 |
| 断幺九 | 1 番 |
| 混一色 | 3 番 |
| 清一色 | 6 番 |
| 七对 | 4 番 |
| 对对胡 | 3 番 |
| 十三幺 | 最大 |

## 基本策略

1. **注意安全牌**：观察已打出的牌，判断哪些牌不会放炮
2. **灵活变通**：根据摸牌灵活调整目标牌型
3. **听牌面广**：尽量让听牌数（可胡的牌）更多
4. **控制风险**：牌局后期谨慎出牌，避免点炮
5. **观察对手**：注意其他玩家吃碰的牌，推断他们的牌型

## 礼仪与规范

- 不偷看他人手牌
- 摸牌和打牌动作不要太慢
- 宣布碰、杠、胡时需及时明确
- 打出的牌按顺序摆放便于确认
