# 德州扑克规则

## 基本设置

| 项目 | 说明 |
|------|------|
| **人数** | 2 ~ 10 人，标准现金局常见 6 人桌或 9 人桌 |
| **牌组** | 标准 52 张扑克牌，无大小王 |
| **目标** | 用 2 张底牌 + 5 张公共牌中任意组合，凑出最强 5 张牌型 |
| **盲注** | 庄家左边第一人为小盲（SB），第二人为大盲（BB）。大盲 = 基准下注单位，小盲 = 大盲的一半 |

## 游戏流程

### ① 发牌前 — Blinds

SB、BB 强制下盲注，每人发 2 张底牌（暗牌）。

### ② 第一轮 — Pre-flop

从 UTG 开始行动，绕一圈到 BB。可选：跟注 / 加注 / 弃牌。

### ③ 翻牌 — Flop

烧 1 张牌，发 3 张公共牌朝上，从 SB 开始新一轮行动。

### ④ 转牌 — Turn

烧 1 张牌，再发 1 张公共牌（共 4 张），再一轮行动。

### ⑤ 河牌 — River

烧 1 张牌，再发 1 张公共牌（共 5 张），最后一轮行动。

### ⑥ 摊牌 — Showdown

最后下注者先亮牌，比 5 张最强组合，最大者赢底池。

## 行动顺序

**翻牌前：** UTG → UTG+1 → HJ → CO → BTN → SB → BB

**翻牌后：** SB → BB → UTG → ... → BTN。**BTN 永远最后行动**，信息最多，是全桌最佳位置。

## 可选行动

| 行动 | 条件 | 说明 |
|------|------|------|
| **Check 过牌** | 本轮无人下注 | 不下注，将行动权传给下一人 |
| **Bet 下注** | 本轮无人下注 | 主动投入筹码，本轮首次下注 |
| **Call 跟注** | 面对下注 | 跟上当前最大下注额，继续参与这局 |
| **Raise 加注** | 面对下注 | 在他人下注基础上再加注，至少要加到上一注的 2 倍 |
| **Fold 弃牌** | 任何时候 | 放弃这局，底牌朝下推出，本轮不再参与 |
| **All-in 全押** | 任何时候 | 将全部剩余筹码投入，即使不够跟注也可操作，会形成边池 |

## 牌型大小（从大到小）

| 排名 | 牌型 | 示例 |
|:---:|------|------|
| 1 | **皇家同花顺** 🏆 | A♠ K♠ Q♠ J♠ 10♠ 同花色 |
| 2 | **同花顺** | 5♣ 6♣ 7♣ 8♣ 9♣ 同花色连续 |
| 3 | **四条** | K♠ K♥ K♦ K♣ 2 |
| 4 | **葫芦** | Q Q Q 7 7（三条 + 一对） |
| 5 | **同花** | A♥ J♥ 8♥ 5♥ 2♥ 同花色不连续 |
| 6 | **顺子** | 4 5 6 7 8 连续不同花 |
| 7 | **三条** | J J J 4 9 |
| 8 | **两对** | A A 8 8 3 |
| 9 | **一对** | K K 7 4 2 |
| 10 | **高牌** | A J 8 5 2 不同花不连续 |

> A 可做最小顺子：A 2 3 4 5（称为"轮子"）。牌型相同时比点数大小，再比踢脚牌（Kicker）。花色不分大小。

## 比牌详细规则

当两名或多名玩家拥有**相同牌型**时：

| 牌型 | 比较规则 |
|------|---------|
| 皇家同花顺 | 永远平局，平分底池 |
| 同花顺 / 顺子 | 比最大的一张牌。A-2-3-4-5 最小 |
| 四条 | 先比四条点数，再比踢脚牌 |
| 葫芦 | 先比三条点数，再比对子点数 |
| 同花 | 从大到小逐张比较 |
| 三条 | 先比三条点数，再依次比踢脚牌 |
| 两对 | 先比大对 → 小对 → 踢脚牌 |
| 一对 | 先比对子点数，再依次比踢脚牌 |
| 高牌 | 从大到小逐张比较 |

五张最佳手牌点数完全相同则平分底池。

## 位置说明（6 人桌）

| 位置 | 说明 |
|------|------|
| **BTN（庄家）** | 最佳位置，翻牌后永远最后行动，信息最充分 |
| **CO（切牌位）** | BTN 左边，位置较好，适合宽范围开牌 |
| **HJ（劫持位）** | 中间位置，偏后，策略灵活 |
| **UTG（枪口位）** | 最差位置，翻牌前第一个行动，信息最少 |
| **SB（小盲）** | 翻牌后第一个行动，位置最劣，但已付出筹码 |
| **BB（大盲）** | 翻牌前最后行动享有 Option 权，翻牌后位置较差 |

**单挑**（2 人局）：庄家下小盲注，翻牌前先行动；对手下大盲注。

## 全押与边池

玩家可以随时将所有剩余筹码**全部押入**。当全押玩家筹码不足时：

1. 用全押玩家的贡献额（其他人等额匹配）形成**主池**
2. 超出部分形成**边池**，只有能覆盖该下注的玩家竞争
3. 全押玩家只能赢取自己参与的底池

> **举例：** 玩家 A（500 筹码）全押，B（1500）跟注，C（1500）跟注。主池 = 500 × 3 = 1500（A/B/C 竞争），边池 = 1000 × 2 = 2000（仅 B/C 竞争）。

## 常用术语

| 术语 | 说明 |
|------|------|
| **3-bet** | 翻牌前第三次加注，代表强牌或极端诈唬 |
| **C-bet** | 续注，翻牌前加注者在翻牌后继续下注维护主动权 |
| **Nuts** | 当前最强的可能牌型，俗称"坚果" |
| **Kicker** | 踢脚牌，牌型相同时用来决定胜负的附带牌 |
| **Bluff** | 诈唬，手牌不强时通过下注逼迫对手弃牌 |
| **Value bet** | 价值注，手牌强时下注让对手以劣势牌跟注 |
| **Pot odds** | 底池赔率，判断是否值得跟注的数学依据 |
| **Run it twice** | 翻两次，All-in 后双方同意将剩余公共牌各跑一次，底池平分 |
| **100BB** | 标准带入筹码量，等于 100 个大盲注，是现金局最常见深度 |

## 更多术语

### 下注术语

| 术语 | 说明 |
|------|------|
| **Check-Raise** | 先过牌诱使对手下注，对手下注后再加注 |
| **4-bet** | 对 3-bet 的再加注 |
| **偷盲 Steal** | 在晚位用较弱的牌加注，企图直接赢得盲注 |
| **挤压 Squeeze** | 前面有加注和跟注后，做出大额 3-bet |
| **Limp** | 翻牌前仅跟注大盲注进入底池（不加注） |
| **弃牌权益** | 通过下注 / 加注逼迫对手弃牌所获得的额外赢率 |

### 策略术语

| 术语 | 说明 |
|------|------|
| **Outs** | 能让你的牌变成最强的剩余牌张数 |
| **听牌 Draw** | 尚未成型但只差一两张牌即可完成的牌型 |
| **半诈唬** | 用听牌下注——即使被跟注仍有机会成牌 |
| **慢打 Slow Play** | 持有强牌但选择不下注 / 小额下注，诱使对手下注 |
| **隐含赔率** | 如果成牌后能从对手处赢取的额外筹码 |
| **TAG** | 紧凶，只打好牌但一旦参与就积极下注 |
| **LAG** | 松凶，参与较多牌局且积极下注 |

### 牌面术语

| 术语 | 说明 |
|------|------|
| **百老汇 Broadway** | A-K-Q-J-10 顺子（最大顺子） |
| **轮子 Wheel** | A-2-3-4-5 顺子（最小顺子） |
| **湿牌面 Wet** | 公共牌有很多听牌可能（连续牌 / 同花） |
| **干牌面 Dry** | 公共牌分散，听牌可能性少 |
| **彩虹面 Rainbow** | 翻牌三种或以上不同花色（无同花听牌） |
| **暗三条 Set** | 底牌对子配合公共牌形成三条 |
| **明三条 Trips** | 一张底牌配合公共牌上的对子形成三条 |

## 底池赔率速算

| Outs | 单轮成牌率 | 两轮成牌率 |
|:---:|:---:|:---:|
| 1 | 2% | — |
| 2 | 4% | — |
| 4 | 9% | 16% |
| 6 | 13% | 24% |
| 8 | 17% | 31% |
| 9 | 19% | 35% |
| 12 | 26% | 45% |
| 15 | 33% | 54% |

**速算法：** 单轮 ≈ outs × 2 + 1；两轮 ≈ outs × 4

**决策公式：** 如果"成牌率 > 跟注所占底池比例"，跟注长期有利可图。

> **举例：** 底池 1000，对手下注 500，需跟注 500 赢 1500。跟注比例 = 500 / 1500 = 33%。如果你有 9 outs（同花听牌，单轮约 19%），单轮不值得跟注；但考虑隐含赔率可能仍有利可图。
