# 德州扑克规则

## 游戏概述

德州扑克（Texas Hold'em）是目前全球最流行的扑克变体。每位玩家持 2 张底牌，与桌上 5 张公共牌组合出最强的 5 张牌型。游戏融合了概率计算、心理博弈和风险控制，规则易学但策略极深。

---

## 游戏组件

- 标准 52 张扑克牌，无大小王
- 筹码
- 2 ~ 10 人，标准现金局常见 6 人桌或 9 人桌

---

## 庄家与盲注

1. 使用**庄家按钮（Button / Dealer）**标记当前庄家位置。每局结束后，按钮顺时针移动一位，小盲、大盲随之轮换，确保所有玩家长期承担相同位置。
2. 按钮左手第一位为**小盲（Small Blind / SB）**，第二位为**大盲（Big Blind / BB）**。
3. 小盲和大盲在发牌前强制下注。大盲为基础下注单位，小盲通常为大盲的一半。
4. 每人发 2 张底牌（暗牌），只有自己可见。

---

# 基础规则

## 游戏流程

每局牌分四轮下注：

| 轮次 | 英文 | 操作 |
|------|------|------|
| 翻牌前 | Pre-flop | 烧 0 张 → 0 张公共牌 → 从 UTG（大盲左手）开始行动 |
| 翻牌 | Flop | 烧 1 张 → 连续翻开 3 张公共牌 → 从 SB 开始行动 |
| 转牌 | Turn | 烧 1 张 → 翻开 1 张公共牌（共 4 张）→ 新一轮行动 |
| 河牌 | River | 烧 1 张 → 翻开最后 1 张公共牌（共 5 张）→ 最后一轮行动 |

所有行动轮结束后进入**摊牌（Showdown）**：

- 若最后一轮有下注且被跟注，**最后一位主动下注或加注者先亮牌**，其余玩家按行动顺序决定是否亮牌（允许弃牌不亮 / Muck）。
- 若最后一轮无人下注，则由按钮左手第一位仍在牌局中的玩家先亮牌。
- 最佳 5 张牌组合赢走底池。

---

## 可选行动

| 行动 | 条件 | 说明 |
|------|------|------|
| **弃牌（Fold）** | 任何时候 | 放弃底牌，退出本局 |
| **过牌（Check）** | 本轮无人下注 | 不下注，行动权传给下一人 |
| **下注（Bet）** | 本轮无人下注 | 主动投入筹码 |
| **跟注（Call）** | 面对下注 | 匹配当前最高下注额 |
| **加注（Raise）** | 面对下注 | 提高下注额。最低加注后的总下注额 = 当前下注额 + 上一轮完整加注幅度 |
| **全押（All-in）** | 任何时候 | 押入全部剩余筹码 |

> ⚠️ **注意**：全押金额不足最低加注幅度时，不会重新开启已行动玩家的加注权。例如 A 下注 100，B 全押 150（不足最低加注），C 跟注后，A 只能跟注不能加注。

---

## 公共牌使用原则

> 💡 **核心原则**：最终牌型必须是恰好 5 张牌，你可以使用 **0、1 或 2 张底牌**与公共牌组合，系统自动从 7 张牌（2 底 + 5 公）中选取最优 5 张组合。

例如你拿 A♥ A♦，公共牌 K♠ K♣ K♥ Q♠ Q♥，你的最佳五张是 **K♠ K♣ K♥ A♥ A♦**（葫芦），而不是 A♥ A♦ K♠ K♣ K♥（两对）。

---

## 牌型大小（从大到小）

| 排名 | 牌型 | 示例 | 说明 |
|:---:|------|------|------|
| 1 | **皇家同花顺** 🏆 | A♠ K♠ Q♠ J♠ 10♠ | 同花的 A-K-Q-J-10 |
| 2 | **同花顺** | 5♣ 6♣ 7♣ 8♣ 9♣ | 同花且连续 |
| 3 | **四条** | K K K K 2 | 四张相同点数 |
| 4 | **葫芦** | Q Q Q 7 7 | 三条 + 一对 |
| 5 | **同花** | A♥ J♥ 8♥ 5♥ 2♥ | 同花色，不连续 |
| 6 | **顺子** | 4 5 6 7 8 | 连续，不同花 |
| 7 | **三条** | J J J 4 9 | 三张相同点数 |
| 8 | **两对** | A A 8 8 3 | 两个对子 |
| 9 | **一对** | K K 7 4 2 | 一个对子 |
| 10 | **高牌** | A J 8 5 2 | 无任何组合 |

> A 可在顺子两端：A-2-3-4-5（轮子 / Wheel）为最小顺子；10-J-Q-K-A（百老汇 / Broadway）为最大顺子。
>
> ⚠️ **花色没有大小之分。** 黑桃不大于红桃。同花之间比点数，不比花色。皇家同花顺之间永远平分。

---

## 比牌规则

牌型相同时，按以下规则比较：

| 牌型 | 比较方法 |
|------|---------|
| 皇家同花顺 | 永远平分底池 |
| 同花顺 / 顺子 | 比最大一张牌。A-2-3-4-5 最小 |
| 四条 | 先比四条点数 → 踢脚牌 |
| 葫芦 | 先比三条点数 → 对子点数 |
| 同花 | 从大到小逐张比较 |
| 三条 | 先比三条点数 → 踢脚牌 |
| 两对 | 大对 → 小对 → 踢脚牌 |
| 一对 | 对子点数 → 踢脚牌 |
| 高牌 | 从大到小逐张比较 |

---

## 平分底池

若两位或以上玩家的最佳五张牌**点数完全一致**，底池平均分配。

> **常见场景：** 公共牌为 A♠ K♥ Q♦ J♣ 10♠，所有仍在牌局中的玩家都拥有 Broadway 顺子，无论底牌是什么，全部平分底池。

---

# 终局与底池

## 全押与边池

玩家可以随时将剩余筹码**全部押入**。当全押玩家筹码不足时：

1. 以全押玩家的贡献额为上限形成**主池**（其他人等额匹配）
2. 超出部分形成**边池**，只有能覆盖该下注的玩家竞争
3. 全押玩家只能赢取自己参与的底池

> **举例：** 玩家 A（500）全押，B（1500）跟注，C（1500）跟注。主池 = 500 × 3 = 1500（A/B/C 竞争），边池 = 1000 × 2 = 2000（仅 B/C 竞争）。

---

## 底池赔率

**底池赔率（Pot Odds）**是判断是否值得跟注的数学依据。

> **公式：** 所需赢率 = 跟注额 ÷（当前底池 + 跟注额）

**举例：** 底池 1000，对手下注 500。你需要跟注 500，跟注后总底池 2000。所需赢率 = 500 ÷ 2000 = **25%**。即你的牌至少要有 25% 的胜率，跟注才有利可图。

**胜率速算表：**

| 补牌数（Outs） | 翻牌 → 转牌 | 翻牌 → 河牌（两轮） |
|:---:|:---:|:---:|
| 4 | 8% | 16% |
| 6 | 13% | 24% |
| 8 | 17% | 31% |
| 9 | 19% | 35% |
| 12 | 26% | 45% |
| 15 | 33% | 54% |

> **速算法：** 翻牌到河牌（两轮）≈ Outs × 4；转牌到河牌（一轮）≈ Outs × 2

# 位置

位置是德州扑克最重要的概念。**越晚行动，信息越多，优势越大。**

## 6 人桌座位

按翻牌后行动顺序，从最早到最晚：

| 位置 | 英文 | 说明 |
|------|------|------|
| **SB（小盲）** | Small Blind | 翻牌后最先行动，位置最差 |
| **BB（大盲）** | Big Blind | 翻牌前最后行动，享有 Option 权 |
| **UTG（枪口位）** | Under the Gun | 翻牌前最先行动，信息最少 |
| **HJ（劫持位）** | Hijack | 中间偏后，策略灵活 |
| **CO（关煞位）** | Cutoff | 庄家左手位，宽范围开池 |
| **BTN（庄位）** | Button | **最佳位置**，翻牌后永远最后行动 |

## 9 人桌对照

| 9 人桌 | 6 人桌 |
|--------|--------|
| UTG | — |
| UTG+1 | — |
| MP（中间位） | — |
| LJ（Lojack） | UTG |
| HJ | HJ |
| CO | CO |
| BTN | BTN |
| SB | SB |
| BB | BB |

> **行动顺序**：翻牌前从 UTG 开始顺时针行动；翻牌后从 SB 开始。BTN 翻牌后永远最后行动。

## 翻前起手牌策略

翻牌前最重要的决策：**RFI（Raise First In）**——前面所有人都弃牌到你，你要不要玩这手牌？

核心原则：**位置越靠后，可以玩的牌越多。** 同样的手牌在 UTG 可以直接弃掉，在 BTN 却可以轻松加注。

本页面配套的**翻前训练器**可以帮你记住各个位置上正确的 RFI 决策。反复练习直到对每个位置的起手牌范围形成直觉。

# 策略建议

以下不是规则，而是帮助提升水平的策略框架。

---

## 位置比手牌更重要

同样的 KJo（K-J 不同花），在 UTG 是**弃牌**，在 BTN 是**轻松加注**。不是牌变了，是你行动时掌握的信息变了。任何时候拿到手牌，先想自己坐在什么位置。

---

## 好牌下注，差牌弃掉

德州扑克的本质是在**有优势时多赢、没优势时少输**。不要每一手牌都跟注"进去看看"。大部分起手牌根本不值得玩，果断弃掉。

---

## 尊重下注背后的信息

对手的每一次下注都在传递信息。翻牌前 3-bet 代表极强的牌力（QQ+、AK 之类）。大额下注通常代表对手有信心。不要轻易"不信邪"——多数时候对手不是在诈唬。

---

## 赔率对了才能跟

做任何跟注决策之前，先算三件事：我需要投入多少？赢下来能拿多少？我的牌有多少概率能成？三者对齐了，跟注才是理性的。

---

## 多练翻前，少纠结翻后

翻牌前决策是基础中的基础。如果你连哪些牌该玩、哪些该弃都拿不准，翻后的复杂局面只会让你更混乱。先通过翻前训练器把 RFI 范围练熟，翻后再慢慢学。

---

# 常见误区

| 误区 | 事实 |
|------|------|
| 必须使用两张底牌 | ❌ 可以使用 0、1 或 2 张底牌组成最佳五张 |
| 黑桃最大，花色有大小 | ❌ 花色没有大小，同花之间比点数 |
| A 只能当最大牌 | ❌ A 也可以当 1，组成 A-2-3-4-5 最小顺子 |
| 同花比顺子"大很多" | ❌ 同花只是牌型等级比顺子高一级，没有"大多少"的概念 |
| 公共牌出皇家同花顺，特定玩家赢 | ❌ 所有人平分底池 |
| 两对 = 四张牌 | ❌ 两对是五张牌中的两组对子 + 一张踢脚牌 |
| 三条一定赢两对 | ✅ 是，牌型等级决定了三条 > 两对 |
| 全押一定会输光 | ❌ 全押只影响自己投入的筹码，不会输掉超过自己带入的量 |

# 常用术语

## 下注术语

| 术语 | 说明 |
|------|------|
| **3-bet** | 第三次下注（盲注→加注→再加注），代表强牌或极化诈唬 |
| **4-bet** | 对 3-bet 的再加注 |
| **C-bet（续注）** | 翻牌前加注者在翻牌圈继续下注，保持主动权 |
| **Check-Raise** | 先过牌诱导对手下注，再加注 |
| **Limp** | 翻牌前仅跟注大盲，不加注（通常不推荐） |
| **Steal（偷盲）** | 在晚位用弱牌加注，试图直接赢下盲注 |
| **Squeeze（挤压）** | 前面有人加注且有人跟注后，做出大额 3-bet |

## 策略术语

| 术语 | 说明 |
|------|------|
| **Outs（补牌）** | 能让你的牌型反超的剩余牌张数 |
| **Equity（权益）** | 手牌在当前局面下获胜的理论概率 |
| **Range（范围）** | 玩家在特定情况下可能持有的所有手牌组合 |
| **Draw（听牌）** | 尚未成型、需要特定牌张才能完成的牌型 |
| **Semi-Bluff（半诈唬）** | 用听牌下注——即使被跟注仍有机会成牌 |
| **Implied Odds（隐含赔率）** | 成牌后预计能从对手处赢取的额外价值 |
| **SPR（筹码底池比）** | Stack-to-Pot Ratio，剩余筹码与底池的比值，影响翻后策略 |
| **Nuts（坚果）** | 当前公共牌面上可能的最强牌型 |
| **Kicker（踢脚牌）** | 牌型相同时决定胜负的附带牌张 |
| **VPIP** | 自愿投入底池率（Voluntarily Put $ In Pot），衡量松紧程度 |

## 牌面术语

| 术语 | 说明 |
|------|------|
| **Broadway（百老汇）** | 10-J-Q-K-A 顺子（最大顺子） |
| **Wheel（轮子）** | A-2-3-4-5 顺子（最小顺子） |
| **Rainbow（彩虹面）** | 翻牌三张不同花色，无同花听牌可能 |
| **Wet Board（湿牌面）** | 公共牌连续或同花，听牌可能多 |
| **Dry Board（干牌面）** | 公共牌分散，听牌可能少 |
| **Set（暗三条）** | 用口袋对子配合公共牌形成三条 |
| **Trips（明三条）** | 用一张底牌配合公共牌上的对子形成三条 |
