# 德州扑克规则

## 概述

德州扑克是最流行的扑克变体，使用一副标准 52 张扑克牌（不含大小王）。每位玩家获得两张私人手牌（底牌），桌面上会公开翻出五张公共牌。玩家从自己的两张底牌和五张公共牌中选出最佳的五张组合来比拼。

## 组件

- 1 副标准 52 张扑克牌（不含大小王）
- 筹码
- 庄家按钮（Dealer Button）

## 位置

| 位置 | 缩写 | 说明 |
|------|------|------|
| 庄家 / 按钮位 | BTN | 发牌者（或由按钮标记）；翻牌后最后行动 |
| 小盲注 | SB | 庄家左侧；下小盲注 |
| 大盲注 | BB | 小盲左侧；下大盲注（最低强制下注） |
| 枪口位 | UTG | 大盲左侧；翻牌前第一个行动 |
| 中间位 | MP | 枪口位到关煞位之间 |
| 关煞位 | CO | 庄家右侧；第二好的位置 |

**单挑**（2人局）：庄家下小盲注，翻牌前先行动；对手下大盲注。

## 准备

1. 确定庄家。每局结束后庄家按钮顺时针轮转。
2. 庄家左侧的两位玩家需要强制下注（盲注）：
   - **小盲注**：通常为大盲注的一半
   - **大盲注**：本局的最低下注额
3. 每位玩家获得 2 张底牌（面朝下）。

## 单牌大小

单牌从大到小排列：**A、K、Q、J、10、9、8、7、6、5、4、3、2**。A 在顺子 A-2-3-4-5（"轮子"）中可作最小牌使用。德州扑克中**花色不分大小**。

## 牌型等级（从高到低）

| 排名 | 牌型 | 说明 | 示例 |
|------|------|------|------|
| 1 | **皇家同花顺** | 同花的 A-K-Q-J-10 | A♠ K♠ Q♠ J♠ 10♠ |
| 2 | **同花顺** | 五张连续的同花牌 | 9♥ 8♥ 7♥ 6♥ 5♥ |
| 3 | **四条** | 四张同点数 + 1 张踢脚牌 | K♣ K♦ K♥ K♠ A♣ |
| 4 | **葫芦** | 三条 + 一对 | Q♦ Q♣ Q♠ 8♥ 8♦ |
| 5 | **同花** | 五张同花色（非连续） | A♦ J♦ 8♦ 6♦ 2♦ |
| 6 | **顺子** | 五张连续的牌（花色不限） | 10♣ 9♦ 8♠ 7♥ 6♣ |
| 7 | **三条** | 三张同点数 + 2 张踢脚牌 | 7♠ 7♥ 7♦ K♣ 3♠ |
| 8 | **两对** | 两个不同的对子 + 1 张踢脚牌 | J♣ J♦ 4♠ 4♥ A♣ |
| 9 | **一对** | 两张同点数 + 3 张踢脚牌 | 9♠ 9♣ K♦ 7♥ 3♠ |
| 10 | **高牌** | 无任何组合，逐张比大小 | A♦ J♣ 8♠ 5♥ 2♣ |

## 比牌详细规则与踢脚牌（Kicker）

当两名或多名玩家拥有**相同牌型**时，按以下规则详细比较：

### 皇家同花顺
永远平局（平分底池）——花色不分大小。

### 同花顺
比较**最大的一张牌**。例：9-8-7-6-5 > 8-7-6-5-4。最小的同花顺是 A-2-3-4-5（A 算最小）。

### 四条
先比**四条的点数**。若相同（公共牌上有四条时可能出现），再比**踢脚牌**（第 5 张牌）。

举例：K-K-K-K-A > Q-Q-Q-Q-A。若双方都是 K-K-K-K，则谁的踢脚牌更大谁赢。

### 葫芦
先比**三条的点数**，再比**对子的点数**。

举例：Q-Q-Q-3-3 > J-J-J-A-A（Q 的三条 > J 的三条）。三条相同时比对子：8-8-8-K-K > 8-8-8-7-7。

### 同花
**从最大到最小逐张比较**。

举例：A♦-J♦-8♦-6♦-2♦ vs A♣-J♣-8♣-5♣-2♣ —— 第四张牌 6 > 5，第一位玩家赢。如果五张牌点数完全相同，则平分底池。

### 顺子
比较**最大的一张牌**。A-K-Q-J-10（百老汇）最大；A-2-3-4-5（轮子）最小。

举例：10-9-8-7-6 > 9-8-7-6-5。最大牌相同则一定平局（如双方都是 Q-J-10-9-8）。

### 三条
先比**三条的点数**，再依次比**踢脚牌**。

举例：7-7-7-A-K > 7-7-7-A-Q（第二张踢脚牌 K > Q）。

### 两对
先比**较大的对子**，再比**较小的对子**，最后比**踢脚牌**。

举例：K-K-5-5-A > K-K-4-4-A（第二对 5 > 4）。两对都相同时：K-K-5-5-J > K-K-5-5-8（踢脚牌 J > 8）。

### 一对
先比**对子的点数**，再依次比**踢脚牌**（最多 3 张）。

举例：9-9-A-K-7 > 9-9-A-Q-J（第二张踢脚牌 K > Q）。

### 高牌
**从最大到最小逐张比较**，直到分出胜负。

举例：A-K-8-5-2 > A-K-7-5-2（第三张 8 > 7）。

### 平分底池
如果最终五张最佳手牌的点数**完全相同**，底池平分。花色永远不会用于决定胜负。

## 游戏流程

每手牌最多四个下注轮。每轮发公共牌前（翻牌前除外），庄家先**烧掉**一张牌（面朝下弃掉），再发公共牌。

### 1. 翻牌前（Pre-Flop）
拿到底牌后，大盲注左侧的玩家（枪口位）首先行动。可选操作：弃牌、跟注（匹配大盲注）、加注。依次顺时针行动。大盲注最后行动，无人加注时可选择过牌或加注。

### 2. 翻牌圈（Flop）
烧 1 张牌，翻开 **3 张公共牌**。从庄家左侧第一位未弃牌的玩家开始下注。此时起玩家可以选择过牌（因为没有强制下注）。

### 3. 转牌圈（Turn）
烧 1 张牌，翻开**第 4 张公共牌**。进行新一轮下注。

### 4. 河牌圈（River）
烧 1 张牌，翻开**第 5 张也是最后一张公共牌**。进行最后一轮下注。

### 5. 摊牌（Showdown）
如果两名或更多玩家未弃牌，则亮出底牌比大小。从 2 张底牌 + 5 张公共牌中取最佳 5 张，牌型最好的玩家赢得底池。

**摊牌顺序：** 最后一个下注或加注的玩家先亮牌。如果河牌圈全部过牌，则庄家左侧第一位玩家先亮牌。被击败的玩家可以选择**盖牌**（不亮牌直接弃掉）。

## 下注规则

### 操作一览
| 操作 | 条件 | 说明 |
|------|------|------|
| **过牌** | 本轮无人下注 | 不下注、不弃牌，行动传给下家 |
| **下注** | 本轮无人下注 | 开启下注（最低 = 大盲注金额） |
| **跟注** | 面对下注 | 匹配当前最高下注额 |
| **加注** | 面对下注 | 提高下注（最低加注额 = 上一次的下注/加注额度） |
| **弃牌** | 任何时候 | 放弃手牌，丧失底池中已投入的所有筹码 |

### 最低加注额
在无限注德州扑克中，加注额必须**至少等于上一次下注或加注的幅度**。例如：大盲注为 100，有人加注到 300（加注幅度为 200），则下一次加注至少要到 500（300 + 200）。

### 下注轮结束
当所有活跃玩家都已经匹配最高下注额或弃牌。如果只剩一名玩家，该玩家立即获胜。

## 全押（All-In）与边池

玩家可以随时将所有剩余筹码**全部押入**。

### 边池规则
当全押玩家的筹码少于完整的下注额时：
1. 用全押玩家的贡献额（其他人等额匹配）形成**主池**。
2. 超出部分形成**边池**，只有能覆盖该下注的玩家竞争。
3. 全押玩家只能赢取自己参与的底池。
4. 多人以不同金额全押时，会形成多个边池。

### 举例
玩家 A（500 筹码）全押。玩家 B（1500）跟注。玩家 C（1500）跟注。
- 主池：500 × 3 = 1500（A、B、C 竞争）
- 边池：1000 × 2 = 2000（仅 B 和 C 竞争）

## 特殊规则

### A（Ace）的用法
- **最大**：A-K-Q-J-10 组成百老汇顺子（Broadway）或皇家同花顺
- **最小**：A-2-3-4-5 组成轮子顺子（Wheel）或最小同花顺（Steel Wheel）
- A-2-3-4-5 是**最小的顺子**；顺子不能"绕回来"（如 Q-K-A-2-3 无效）

### 底牌的使用
你可以使用**两张、一张或零张**底牌与公共牌搭配。系统自动从 7 张可用牌中选出最佳 5 张。如果 5 张公共牌本身就是所有玩家的最佳手牌，则平分底池（俗称"打公共牌"）。

### 桌面筹码规则
玩家只能使用发牌时已在桌面上的筹码。一手牌进行中不能额外购买筹码。

### 分次下注限制
加注必须**口头宣告**或**一次性推出筹码**。未经宣告的分次放入筹码（String Bet）通常不被允许。

## 术语表

### 基本术语

| 术语 | 英文 | 说明 |
|------|------|------|
| 底池 | Pot | 当前牌局中所有已下注筹码的总和 |
| 底牌 / 手牌 | Hole Cards | 每位玩家私有的 2 张面朝下的牌 |
| 公共牌 | Community Cards | 桌面上面朝上的 5 张共享牌 |
| 盲注 | Blind | 强制下注（小盲 + 大盲），保证每手有底池 |
| 前注 | Ante | 所有玩家在发牌前缴纳的强制小额下注（锦标赛后期常见） |
| 烧牌 | Burn Card | 发公共牌前面朝下弃掉的一张牌，防止作弊 |
| 亮牌 | Showdown | 最终翻开底牌比牌的环节 |
| 盖牌 | Muck | 被击败时选择不亮出底牌直接弃牌 |
| 踢脚牌 | Kicker | 决定同牌型胜负的辅助牌 |

### 下注术语

| 术语 | 英文 | 说明 |
|------|------|------|
| 过牌-加注 | Check-Raise | 先过牌（诱使对手下注），对手下注后再加注 |
| 持续下注 | C-bet (Continuation Bet) | 翻牌前加注的玩家在翻牌后继续下注（无论牌力） |
| 3-bet | 3-Bet | 对一个加注的再加注（翻牌前：盲注→加注→3-bet） |
| 4-bet | 4-Bet | 对 3-bet 的再加注 |
| 偷盲 | Steal | 在晚位用较弱的牌加注，企图直接赢得盲注 |
| 挤压 | Squeeze | 前面有加注和跟注后，做出大额 3-bet |
| 过牌 | Limp | 翻牌前仅跟注大盲注（不加注）进入底池 |
| 弃牌权益 | Fold Equity | 通过下注/加注逼迫对手弃牌所获得的额外赢率 |

### 策略术语

| 术语 | 英文 | 说明 |
|------|------|------|
| 坚果 | Nuts | 当前牌面上可能的最强牌型 |
| 出牌数 | Outs | 能让你的牌变成最强的剩余牌张数 |
| 听牌 | Draw | 尚未成型但只差一两张牌即可完成的牌型 |
| 诈唬 | Bluff | 用弱牌大额下注企图逼对手弃牌 |
| 半诈唬 | Semi-Bluff | 用听牌下注——即使被跟注仍有机会成牌 |
| 价值下注 | Value Bet | 认为自己牌力最强时下注，希望对手跟注 |
| 慢打 | Slow Play | 持有强牌但选择不下注/小额下注，诱使对手下注 |
| 底池赔率 | Pot Odds | 跟注金额与底池的比率，用于判断跟注是否有利可图 |
| 隐含赔率 | Implied Odds | 如果成牌后能从对手处赢取的额外筹码 |
| 位置优势 | Position | 后行动的玩家信息更多，通常更有利 |
| 紧凶 | TAG (Tight-Aggressive) | 只打好牌但一旦参与就积极下注的风格 |
| 松凶 | LAG (Loose-Aggressive) | 参与较多牌局且积极下注的风格 |

### 牌面术语

| 术语 | 英文 | 说明 |
|------|------|------|
| 百老汇 | Broadway | A-K-Q-J-10 顺子（最大顺子） |
| 轮子 | Wheel | A-2-3-4-5 顺子（最小顺子） |
| 对子面 | Paired Board | 公共牌中有对子（有人可能有葫芦/四条） |
| 湿牌面 | Wet Board | 公共牌有很多听牌可能（连续牌/同花） |
| 干牌面 | Dry Board | 公共牌分散，听牌可能性少 |

| 暗三条 | Set | 底牌对子配合公共牌形成三条 |
| 明三条 | Trips | 一张底牌配合公共牌上的对子形成三条 |

## 底池赔率速算

听同花（翻牌后 9 张 outs）：翻牌到河牌约 **35%** 成牌率  
听顺子（8 张 outs）：翻牌到河牌约 **31%**  
翻牌到转牌单张 out：约 **2.1%**（outs × 2 + 1 速算法）

| 转牌/河牌单轮 | Outs | 约成牌率 |
|:---:|:---:|:---:|
| | 1 | 2% |
| | 2 | 4% |
| | 4 | 9% |
| | 6 | 13% |
| | 8 | 17% |
| | 9 | 19% |
| | 12 | 26% |
| | 15 | 33% |

**决策公式：** 如果"底池赔率 > 成牌率"，跟注长期有利可图。

例：底池 1000，对手下注 500，需要跟注 500 赢 1500。底池赔率 = 500/1500 = 33%。如果你有 9 outs（同花听牌，约 19%），单轮不值得跟注；但考虑隐含赔率（成牌后可赢更多），可能仍有利可图。
