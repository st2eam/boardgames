# UNO Flip 规则

## 概述

UNO Flip 是经典 UNO 的变体，采用双面卡牌设计。牌堆分为**光明面**（温和，类似经典 UNO）和**黑暗面**（惩罚性更强）。玩家需要在游戏中翻转牌堆，并率先出完所有手牌。

## 组件

- 112 张双面卡牌（光明面 / 黑暗面）
- 每面有各自的一套功能牌

## 准备

1. 选定庄家。牌堆以光明面朝上洗匀。
2. 每位玩家发 7 张牌，光明面朝上。
3. 剩余牌堆放在中央作为抽牌堆。
4. 翻开最上面一张牌作为弃牌堆起点。如果是功能牌，执行其效果。

## 光明面卡牌

| 卡牌 | 效果 |
|------|------|
| **罚抽一张**（蓝/绿/黄/红） | 下家抽 1 张牌并跳过回合 |
| **反转** | 改变出牌方向 |
| **跳过** | 下家跳过回合 |
| **翻转** | 所有玩家将牌堆和手牌翻至黑暗面 |
| **万能牌** | 指定下一种颜色 |
| **万能罚抽两张** | 指定颜色，下家抽 2 张并跳过回合 |

## 黑暗面卡牌

| 卡牌 | 效果 |
|------|------|
| **罚抽五张** | 下家抽 5 张牌并跳过回合 |
| **反转** | 改变出牌方向 |
| **跳过所有人** | 其他所有玩家跳过回合 |
| **翻转** | 所有玩家将牌堆和手牌翻回光明面 |
| **万能牌** | 指定下一种颜色 |
| **万能罚抽颜色** | 下家需一直抽牌直到抽到指定颜色，然后跳过回合 |

## 游戏流程

1. 轮到你时，打出一张与弃牌堆顶部**颜色**、**数字**或**符号**匹配的牌。
2. 如果没有可出的牌，抽 1 张牌。如果抽到的牌符合出牌条件，可立即打出。
3. 打出倒数第二张牌时，必须在下一轮开始前喊出 **"UNO!"**。如果被他人抢先指出未喊 UNO，罚抽 2 张牌。
4. 翻转牌会将整个游戏在光明面和黑暗面之间切换。翻转后，手牌和牌堆全部翻面。

## 获胜条件

率先出完手牌的玩家赢得该轮。根据对手剩余手牌计分：

| 卡牌类型 | 分值 |
|---------|------|
| 数字牌 | 面值 |
| 罚抽一张 / 罚抽五张 | 10 |
| 反转 / 跳过 / 跳过所有人 / 翻转 | 20 |
| 万能牌 | 40 |
| 万能罚抽两张 / 万能罚抽颜色 | 50 |

先累计达到 500 分的玩家赢得整局游戏。
