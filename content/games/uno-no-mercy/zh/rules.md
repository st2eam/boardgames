# UNO Show 'Em No Mercy 规则

## 概述

UNO Show 'Em No Mercy 是美泰（Mattel）出版的最残忍版 UNO。拥有 168 张牌、巨额抽牌惩罚、叠加规则、换手机制和淘汰规则。两种获胜方式：出完手牌或淘汰所有其他玩家。

## 组件

168 张牌：

| 卡牌类型 | 数量 |
|---------|------|
| 数字牌（0-9，4 色） | 80 |
| 弃同色牌（4 色） | 12 |
| 跳过（4 色） | 12 |
| 反转（4 色） | 12 |
| 罚抽 2 张（4 色） | 8 |
| 罚抽 4 张（4 色） | 8 |
| 跳过所有人（4 色） | 8 |
| 万能颜色轮盘 | 8 |

| 万能罚抽 6 张 | 4 |
| 万能罚抽 10 张 | 4 |

## 准备

1. 充分洗匀牌堆。
2. 每位玩家发 **7 张**牌。
3. 剩余牌面朝下作为抽牌堆。
4. 翻开顶部第一张作为弃牌堆起点。

## 游戏流程

打出与弃牌堆顶部**颜色**、**数字**或**符号**匹配的牌。无法出牌时抽 1 张（匹配可立即打出）。剩 1 张牌时必须喊 **"UNO!"**——被指出未喊则罚抽 **4 张**。

## 特殊规则

### 叠加规则
罚抽牌**可以叠加**。有人对你打出罚抽牌（+2、+4、+6、+10）时，你可以打出**相同或更高数值**的罚抽牌将惩罚传给下家。总数持续累加，直到有人无法叠加，该玩家必须抽取**所有**累计的牌。

### 7 换手
打出 **7**（任意颜色）时，**必须**与另一位玩家交换全部手牌。

### 0 传手
打出 **0**（任意颜色）时，**所有玩家**必须将全部手牌传给当前出牌方向的下一位玩家。

### 无情规则（淘汰）
任何时刻手牌达到 **25 张或以上**的玩家**立即被淘汰**。其手牌暂放一旁，待抽牌堆用完时重新洗入。

## 功能牌

| 卡牌 | 效果 |
|------|------|
| **跳过** | 下家跳过回合 |
| **反转** | 改变出牌方向 |
| **罚抽 2 张（+2）** | 下家抽 2 张并跳过回合。可叠加。 |
| **罚抽 4 张（+4）** | 下家抽 4 张并跳过回合。可叠加。 |
| **弃同色牌** | 弃掉手中所有与此牌同色的牌 |
| **跳过所有人** | 其他所有玩家跳过回合，你立即再出一个回合 |
| **万能颜色轮盘** | 指定一种颜色。下家从牌堆持续翻牌，直到翻出指定颜色为止 |
| **万能反转罚抽 4 张** | 反转方向，且新的下家抽 4 张 |
| **万能罚抽 6 张** | 指定颜色，下家抽 6 张并跳过回合 |
| **万能罚抽 10 张** | 指定颜色，下家抽 10 张并跳过回合 |

## 获胜条件

两种获胜方式：
1. **出完手牌** — 率先打完所有手牌
2. **最后存活** — 其他所有玩家被无情规则淘汰
