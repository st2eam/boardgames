# UNO 规则

## 概述

UNO 是经典的家庭卡牌游戏。玩家轮流打出手牌，需要与弃牌堆顶部的牌在颜色、数字或符号上匹配。率先清空手牌即可获胜——但当手牌只剩最后一张时，别忘了大喊"UNO！"

## 组件

112 张牌（经典版 108 张 + 4 张新万能牌）：

| 颜色 | 数字牌 | 功能牌 |
|------|--------|--------|
| 蓝 | 1× 0，2× 各 1-9 | 2× 跳过，2× 反转，2× +2 |
| 绿 | 1× 0，2× 各 1-9 | 2× 跳过，2× 反转，2× +2 |
| 红 | 1× 0，2× 各 1-9 | 2× 跳过，2× 反转，2× +2 |
| 黄 | 1× 0，2× 各 1-9 | 2× 跳过，2× 反转，2× +2 |
| 万能 | — | 4× 万能牌，4× 万能 +4，1× 万能交换手牌 或 万能重洗手牌，3× 万能自定义牌 |

## 准备

1. 洗匀牌堆。
2. 每位玩家发 **7 张**牌。
3. 剩余牌面向下作为抽牌堆。
4. 翻开最上面一张牌作为弃牌堆起点。

**首张牌特殊规则：**
- **万能牌** — 起始玩家选择颜色开始游戏。
- **万能 +4** — 洗回牌堆重新翻。
- **跳过** — 起始玩家（庄家左侧）被跳过。
- **反转** — 庄家先出牌，方向变为逆时针。
- **+2** — 起始玩家抽 2 张并跳过回合。

## 游戏流程

轮到你的回合时，打出**一张**与弃牌堆顶部匹配的手牌：
- **颜色** — 颜色相同
- **数字** — 数字相同
- **符号** — 功能符号相同
- **万能牌** — 随时可打出

如果无法出牌（或不想出），从抽牌堆抽**一张**。抽到的牌若符合条件可立即打出，否则回合结束。

> 每回合只能出**一张**牌。不能在同一回合叠加多张功能牌。

### 喊 UNO!

打出**倒数第二张**手牌时（手中只剩一张时），必须大声喊出 **"UNO!"**。如果在下一位玩家出牌前被其他玩家指出未喊，罚抽 **2 张**。每次回到只剩一张牌的状态都需要再次喊 UNO。

## 功能牌

| 卡牌 | 效果 |
|------|------|
| **跳过** | 下一位玩家跳过回合 |
| **反转** | 改变出牌方向（2 人局效果等同跳过） |
| **+2** | 下家抽 2 张并跳过回合 |
| **万能牌** | 指定接下来的颜色，可在任何时候打出 |
| **万能 +4** | 指定颜色，下家抽 4 张并跳过回合 |
| **万能交换手牌** | 指定颜色，并与任一玩家交换全部手牌 |
| **万能重洗手牌** | 指定颜色，收集所有人手牌洗匀后重新平均分配 |
| **万能自定义牌** | 指定颜色，效果由玩家开局前协商决定 |

### +4 质疑规则

万能 +4 只能在你**没有当前颜色牌**时打出。如果被质疑：
1. 下家可选择接受或质疑。
2. 若被质疑，你必须展示手牌。
3. 你**有**当前颜色的牌 → **你**抽 4 张作为惩罚。
4. 你**没有**当前颜色的牌 → 质疑者抽 **6 张**作为惩罚。

### 万能交换手牌

将你的手牌与你选定的任一玩家交换。如果这是你的最后一张牌，当作普通万能牌打出即可结束游戏（不执行交换）。

### 万能重洗手牌

收集**所有玩家**的手牌，洗匀后从打出者左侧开始逐张重新分配，直到牌发完。部分玩家可能多/少一张。如果是最后一张牌，当作普通万能牌结束游戏。

## 双人规则

2 人对局时规则微调：
- **反转** → 效果等同跳过（对手跳过回合）
- **跳过** → 打出后可以立即再出一张牌
- **+2 / 万能 +4** → 对手抽牌后，轮次立即回到你

## 计分

根据所有对手剩余手牌计分：

| 卡牌类型 | 分值 |
|---------|------|
| 数字牌 0-9 | 面值 |
| 跳过 / 反转 / +2 | 20 |
| 万能牌 / 万能 +4 | 50 |
| 万能交换手牌 / 万能自定义牌 | 40 |

率先累计达到 **500 分**的玩家赢得整场游戏。

## 抽牌堆用完

如果抽牌堆耗尽且无人获胜，将弃牌堆（保留顶牌）洗匀翻面作为新的抽牌堆继续游戏。
